# ⚔️ Jangre — 텍스트 RPG

> 선택이 결과를 바꾸는 Flutter 텍스트 RPG

## 소개

모바일에서 5분 안에 한 판 완결되는 턴제 텍스트 RPG.  
선택지에 따라 3가지 이상의 다른 엔딩에 도달한다.

## 주요 기능

- ⚔️ **턴제 전투** — 공격·방어·아이템 행동
- 📖 **스토리 분기** — 선택지에 따른 다른 전개
- 🎒 **인벤토리** — 아이템 획득 및 전투 중 사용
- 🏆 **다중 엔딩** — 승리 / 패배 / (히든)

## 실행 방법

```bash
flutter pub get
flutter run
```

자세한 내용은 [docs/setup.md](docs/setup.md) 참조.
