# AI Agent 워크플로우 패턴

## 001 — 문서 우선 생성 패턴 (2026-04-28)

### 패턴
코드 작성 전에 문서부터 AI Agent로 생성 → 검토 → 코드 구현

### 단계
1. 비전/요구사항 문서 생성
2. WBS·일정 생성
3. 아키텍처 설계
4. setup/deploy/testing 문서
5. 코드 구현
6. 발표 자료 생성

### 효과
- 방향이 명확해져 구현 오류 감소
- 발표 Q&A 대비 문서가 자동으로 완성됨
- 과제 5점 + 가산점 동시에 충족 가능

## 002 — 체크리스트 기반 구현 패턴 (2026-05-26)

### 패턴
슬라이드의 체크리스트 항목을 그대로 AI에게 전달 → 항목별 순서 구현

### 단계
1. 수업 슬라이드에서 체크리스트 추출
2. 각 항목을 AI에게 순서대로 지시
3. 완료 즉시 다음 항목으로 이동

### 효과
- 수업 기준과 100% 일치하는 산출물 생성
- 놓치는 항목 없음

## 003 — 병렬 작업 패턴 (2026-05-26)

### 패턴
독립적인 여러 파일을 동시에 AI Agent에게 생성 요청

### 예시
```
다음 파일들을 동시에 작성해줘:
- docs/deploy.md (배포 가이드)
- docs/testing.md (테스트 가이드)
- notes/patterns/... (LLM Wiki 항목)
```

### 효과
- 순차 작업 대비 약 3배 빠름
- AI 컨텍스트 안에서 일관성 유지
