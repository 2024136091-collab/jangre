# LLM Wiki — Jaeho의 암묵지 노트

AI Agent 작업 중 알게 된 노하우를 카테고리별로 정리합니다.

## 카테고리

| 폴더 | 내용 |
|---|---|
| `prompts/` | 잘 먹힌 프롬프트 패턴 / 안 먹힌 이유 |
| `failures/` | 실패 사례와 원인 분석 |
| `patterns/` | 반복적으로 유용한 패턴 |
| `tools/` | 도구별 차이점 및 팁 |

## 가산점 인정 기준 (항목 C)

- 최소 10개 이상 항목
- 최근 6개월 이내 갱신 흔적 (커밋 로그)
- 실제 작업과 연결된 사례 최소 3개
