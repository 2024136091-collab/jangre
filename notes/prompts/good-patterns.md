# 잘 먹힌 프롬프트 패턴

## 001 — 부트스트랩 프롬프트 (2026-04-28)

### 상황
바이브코딩 프로젝트 문서 구조 전체를 한 번에 생성해야 함

### 프롬프트 패턴
```
나는 [주제]를 만들고 싶다. 다음을 수행해줘.
1) 비전과 목표를 .planning/00-vision.md에 작성
2) 핵심 사용자 시나리오 3개를 .planning/01-requirements.md에 작성
3) 기능을 MoSCoW로 분류
4) 모호한 부분은 선택형 질문으로 물어본 후 작성
플랫폼은 모바일 앱으로 추천 1개와 이유를 제안해.
```

### 결론
- 단계별로 명확히 지시하면 일관된 문서 생성 가능
- MoSCoW 분류 요청 시 우선순위가 명확해짐
