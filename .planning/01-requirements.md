# 01-requirements.md — 기능 요구사항 (MoSCoW)

## 핵심 사용자 시나리오

### 시나리오 1 — 탐험가 (스토리 진행)

- **사용자**: 텍스트 어드벤처를 좋아하는 20대
- **상황**: 이동 중 잠깐 게임을 즐기고 싶다
- **행동**: 캐릭터를 생성하고 스토리 선택지를 고르며 진행한다
- **결과**: 선택에 따라 스탯·플래그가 변하고 다른 엔딩에 도달한다
- **관련 기능**: M01 · M02 · M03 · M04 · M11

### 시나리오 2 — 전사 (전투 수행)

- **사용자**: 전략적 전투를 즐기는 게이머
- **상황**: 스토리 중 몬스터와 조우한다
- **행동**: 턴제 전투에서 공격·스킬·아이템·도망을 선택하며 상태이상을 관리한다
- **결과**: 전투 승리 후 XP·아이템을 획득하고 스토리로 복귀한다
- **관련 기능**: M05 · M06 · M07 · M08

### 시나리오 3 — 수집가 (성장·장비)

- **사용자**: 캐릭터 육성을 좋아하는 RPG 팬
- **상황**: 레벨업 후 스탯을 분배하고 새 장비를 장착하고 싶다
- **행동**: 획득한 장비를 인벤토리에서 확인하고, 스탯에 맞는 장비를 장착한다
- **결과**: 강화된 캐릭터로 더 어려운 구간에 도전한다
- **관련 기능**: M08 · M09 · M10 · S01

## Must — 필수 구현

| ID | 기능 | 설명 |
|---|---|---|
| M01 | 캐릭터 생성 | 이름 입력, 클래스 선택 (전사·마법사·도적) |
| M02 | 캐릭터 스탯 | HP·MP·STR·DEF·AGI·LV·XP 관리 |
| M03 | 텍스트 스토리 화면 | 지문 출력 + 선택지 2~4개 |
| M04 | 스토리 분기 | 조건부 선택지 (스탯 임계값·보유 아이템·플래그 체크) |
| M05 | 턴제 전투 | 공격·스킬·아이템·도망 선택 |
| M06 | 데미지 공식 | STR·DEF·AGI 반영, 치명타 확률 |
| M07 | 상태이상 | 독(매 턴 HP 감소)·스턴(1턴 행동 불가)·화상(MP 감소) |
| M08 | 인벤토리 | 아이템 획득·사용·버리기, 최대 20슬롯 |
| M09 | 장비 시스템 | 무기·방어구 슬롯, 장착 시 스탯 변동 |
| M10 | 저장/불러오기 | 슬롯 3개, Hive 기반 오프라인 저장 |
| M11 | 다중 엔딩 | 5종 이상, 선택 이력·스탯에 따라 분기 |

| ID | 기능 | 설명 |

| S02 | 상점 시스템 | 골드 화폐, 아이템 구매·판매 |
