저의 바이브 코딩 프로젝트 JANGRE 중간발표를 시작하겠습니다.
JANGRE는 Flutter로 개발 중인 모바일 텍스트 RPG입니다.

------------------------------------------------

[슬라이드 2] 0:15 ~ 0:50  문제 정의 · 비전
기존 모바일 RPG에는 4가지 문제가 있습니다.
과금과 광고가 중심이고, 끝이 없어서 짧은 시간에 완결할 수 없습니다.
선택해도 결과가 같고, 오프라인도 안 됩니다.

JANGRE의 비전은 이 문제를 해결하는 것입니다.
선택이 캐릭터를 성장시키고 세계를 바꾸는 텍스트 RPG.
5~10분 안에 완결되고, 엔딩은 5종, 오프라인에서도 완전히 실행됩니다.

------------------------------------------------

[슬라이드 3] 0:50 ~ 1:20  WBS
7주 개발 일정입니다.
9주차 기획과 11주차 Must 기능 구현은 이미 완료했습니다.
캐릭터 스탯, 스토리 분기, 턴제 전투, 인벤토리, 다중 엔딩 3종.
목표가 50% 달성이었는데 100% 완료했습니다.
현재 10주차 설계 단계이며, 이후 저장 기능과 테스트를 진행할 예정입니다.

------------------------------------------------

[슬라이드 4] 1:20 ~ 1:55  아키텍처
아키텍처는 4레이어로 설계했습니다.
Presentation 레이어에 Flutter 위젯 7개,
Application 레이어에 Riverpod으로 상태 관리,
Domain 레이어에 전투와 스토리 로직을 순수 Dart로 작성해 단위 테스트가 가능하고,
Data 레이어에 Hive로 오프라인 저장 슬롯 3개를 구현합니다.
