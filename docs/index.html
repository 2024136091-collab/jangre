기획·설계 병목

📄

문서화 부재

코드는 짜지만 아키텍처 문서, 설정 가이드, 발표 자료는 별도로 다시 만들어야 합니다.

🔁

반복 작업

프로젝트마다 같은 구조를 다시 세팅합니다. 창의적인 작업 시간이 사라집니다.

🎯

방향 흔들림

기획 없이 코드부터 짜면 기능이 늘어날수록 방향을 잃고 리팩터링이 반복됩니다.

🚀
