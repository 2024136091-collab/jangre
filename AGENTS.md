# AGENTS.md — AI Agent 활용 지침

## 이 프로젝트에서 AI Agent를 사용하는 방법

### 사용 도구
- **Claude Code (Antigravity)** — 주력 AI Agent

### 활용 영역

| 영역 | AI 역할 | 사람 역할 |
|---|---|---|
| 기획 문서 | 초안 생성 | Must/Should 결정, 검토 |
| 코드 생성 | 구현 | 로직 검토, 밸런스 조정 |
| 아키텍처 | 설계 제안 | 최종 결정 |
| 문서화 | 작성 | 정확성 검토 |
| 발표 슬라이드 | 초안 | 내용 검토, 발표 연습 |

### AI에게 잘 작동한 프롬프트 패턴

```
[컨텍스트] 이 프로젝트는 Flutter 텍스트 RPG입니다.
[요청] [구체적인 기능] 을 구현해주세요.
[제약] Must 기능 우선, 단일 파일 최대 200줄.
[확인] 구현 후 어떤 부분을 제가 검토해야 하는지 알려주세요.
```

### 주의사항

- AI가 생성한 스탯 수치는 직접 플레이하며 검토할 것
- 스토리 분기 흐름은 종이에 먼저 그린 후 AI에게 구현 요청
- AI가 모른다고 할 때는 더 작은 단위로 쪼개서 요청
